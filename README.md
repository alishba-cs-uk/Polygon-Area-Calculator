# Polygon Area Calculator

## Overview
The Polygon Area Calculator is a Python project that calculates the area, perimeter, and diagonal of geometric shapes such as rectangles and squares.

## Features
- Create rectangle and square objects
- Calculate area
- Calculate perimeter
- Calculate diagonal
- Generate a visual representation of the shape
- Calculate how many times one shape fits inside another

## Technologies Used
- Python
